# Guardiant Website

The marketing site for [Guardiant](https://github.com/paarthbhatt/Guardiant) — the first security scanner built for AI-generated code.

## Pages

| Route | Description |
|---|---|
| `/` | Landing page with agent library, comparison table, 5-phase engine |
| `/product` | Deep dive into features, agents, and detection capabilities |
| `/research` | VCVF, CVC, and TIEF framework documentation |
| `/use-cases` | Real-world scan examples and vulnerability patterns |

## Stack

- **Next.js 14** — static export, no server runtime
- **Tailwind CSS** — design tokens, custom properties, dark mode
- **Framer Motion** — scroll-triggered animations, stagger effects
- **Three.js** + **React Three Fiber** — 3D hero elements
- **Lucide React** — icon set

## Development

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Build

```bash
npm run build
npm start
```
